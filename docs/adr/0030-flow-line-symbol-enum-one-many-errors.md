# Перечисление символа FlowLine — `ONE_ERROR`/`MANY_ERRORS`; тир «warning» упразднён

> Статус: принят · 2026-07-18 · Реализация: в коде

## Context

Значение `ERROR` перечисления `FLOW_LINE_SYMBOL_TYPES` (`src/interfaces/types.ts`) означает «символ введён неверно один раз» и красится приглушённой горчицей (мягкий тир). Значение `ERRORS` — «введён неверно несколько раз» — красный. Но слово «error» во всей остальной системе означает красный / danger: роли `--color-error*`, `KeyCapPressResult.ERROR`, `FingerNavigationRole.ERROR`. Одно слово несёт два цвета.

Плюс различие `ERROR` / `ERRORS` — это одна буква `s`, её легко проглядеть: значения-соседи в одном перечислении отличаются суффиксом, а не корнем, и на месте вызова читаются как опечатка друг друга.

## Decision

Переименовать значения перечисления: `ERROR → ONE_ERROR`, `ERRORS → MANY_ERRORS`. Доменная модель: и единичный, и повторный промах — это **ошибки**, разница только в количестве.

Отдельный «warning» (предупреждение) как тир **упраздняется**: единичная ошибка подаётся мягче цветом, множественная — жёстче. Слово «warning» / «предупреждение» уходит из символьной семантики; горчица `--mustard` — это «цвет единичной ошибки», а не «предупреждения».

(`MANY` для двух промахов — устоявшийся разговорный термин, принят намеренно.)

## Scope

Затрагивается только цепочка `FlowLineSymbolType`:

- тип и перечисление `FLOW_LINE_SYMBOL_TYPES` (`src/interfaces/types.ts`);
- `getSymbolType` (`src/lib/typing-stream.ts`);
- компонент `RegularSymbol` (CSS-классы и токены);
- символьные роли `--color-symbol-error` / `--color-symbol-errors`.

**НЕ затрагивается** `KeyCapPressResult.ERROR`, `FingerNavigationRole.ERROR`, а также фон-вспышка строки при ошибке нажатия (`--flow-line-*-background`, `--color-*-dim`, `--color-symbol-*-bg` в старых темах) — это другое перечисление (результат нажатия), с ним «error» = красный остаётся.

## Consequences

- Токены и CSS-классы переименованы (см. фазу B плана исправлений).
- «error» теперь означает красный / danger единообразно по всей системе.
